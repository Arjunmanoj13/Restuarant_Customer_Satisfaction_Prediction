# Restaurant Customer Satisfaction Prediction

## Project Overview
This project predicts whether a restaurant customer is highly satisfied based on their dining behavior and preferences. By analyzing factors such as age, income, visit frequency, spending patterns, and preferred cuisine, the model helps understand customer satisfaction levels.

## Technologies Used
- Python
- Machine Learning
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Dataset
The dataset includes the following features:

- CustomerID
- Age
- Gender
- Income
- VisitFrequency
- AverageSpend
- PreferredCuisine
- Satisfaction Ratings

Target Variable:
- **HighSatisfaction** (indicates whether the customer is highly satisfied or not)

## Methodology
1. Data preprocessing and cleaning  
2. Feature encoding and transformation  
3. Training machine learning models  
4. Evaluating model performance using accuracy and confusion matrix

## Model Used
- Logistic Regression
- Perceptron
- KNN

## Result
The model predicts the probability of high customer satisfaction based on dining patterns and demographic features.
